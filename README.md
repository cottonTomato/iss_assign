# Full Stack Programming Assignment (Java, Web, DBMS, Python)

## Overview

This repository contains the solutions and hands-on implementations for the **Full Stack / Pre-Joining Training assignment**, covering:

- Core Java concepts (core_java)
- Client-side programming using HTML, CSS & JavaScript (basic_form)
- Database and SQL fundamentals (basic_sql)
- Java Web Application using JSP, Servlets, Maven, and Tomcat (example_e2e)
- Basic Python programming (python)

The implementation follows the syllabus and guidelines provided in the assignment PDFs .
